## Раунд 5 — Предварительное голосование (ТОП-20)

Место: 1
ID идеи: R1-1-03
Идея: Сквозная модель оптимизации от решения о начале работ до физического пуска — с доработкой: сегменты с «точками фиксации» (milestones), Monte Carlo между соседними вехами, сохранение межпериодных связей без размывания дисперсии
Почему: Наивысшая стратегическая ценность — это единственная идея, которая формализует связь между решениями подготовительного периода и результатом основного. Доработка по итогам критики C-6-01 (точки фиксации) сделала идею реалистичной. Позволяет количественно обосновать концепцию сооружения для стандарта подготовительного периода.

Место: 2
ID идеи: R1-1-02
Идея: Оптимизация критического пути основного периода с акцентом на ТМР — полная модель с PERT, Monte Carlo, итеративной оптимизацией, ML на данных завершённых блоков, многокритериальной оптимизацией F(Γ) = [T, C, R, Q]
Почему: ТМР — доказанный главный ограничитель сроков основного периода. Прямое воздействие на длительность сооружения. Модель применяется комплексно, данные доступны с завершённых блоков ВВЭР.

Место: 3
ID идеи: R1-1-01
Идея: Вероятностная оценка и оптимизация графика подготовительного периода до первого бетона — с доработкой: байесовский подход для работ с малой выборкой, широкие prior для регуляторных процессов
Почему: Базовое применение модели к подготовительному периоду (2–8 лет неопределённости). Доработка с байесовским подходом решает проблему нехватки данных. Прямо отвечает задаче формирования стандарта подготовительного периода.

Место: 4
ID идеи: R1-1-20
Идея: Количественная оценка готовности проекта к строительству (Construction Readiness) — каждое направление готовности как ветвь GERT-графа, Monte Carlo для вероятности достижения готовности к целевой дате, критерий Вальда для наихудшего направления
Почему: Переводит оценку готовности из качественной в количественную плоскость. Непосредственно востребовано при принятии решения о первом бетоне. Комплексно задействует все компоненты модели.

Место: 5
ID идеи: R1-1-04
Идея: Оптимизация цепочки поставок ОДЦИ — PERT для производственных неопределённостей, GERT для альтернативных поставщиков, критерий Сэвиджа для выбора стратегии закупок, ML для предсказания задержек
Почему: Срывы поставок ОДЦИ — один из главных рисков любого проекта АЭС. Идея сохраняет системный характер, данные о поставщиках накапливаются Росатомом. Доработка по R1-8-04 (критерий Вальда для ОДЦИ, JIT только для массового оборудования) усиливает применимость.

Почему: ПНР — этап с наибольшей долей неопределённости и частыми выходами за плановые сроки. GERT-моделирование возвратных петель — прямое и естественное применение модели. ML на данных ПНР предыдущих блоков реалистичен.

Место: 7

Почему: Задержка выпуска РД — хроническая и повсеместная проблема, каскадно задерживающая строительство. Идея связывает проектирование со строительством в единую модель. Доработка R1-8-06 (ограничение на ресурсы института) усиливает реализуемость.

Почему: Планирование «от обратного» — сильный производственный подход. Идея решает конкретную дилемму (временные схемы vs ожидание постоянных), которая реально возникает на каждом проекте. Хорошо использует деревья решений из модели.

Почему: Уникальное преимущество Росатома — серийность. Идея позволяет количественно планировать эффект серийности и оптимальное перекрытие блоков. Доработка R1-5-06 (коэффициент затухания организационной памяти) учитывает реальность текучести кадров.

Почему: Моя доработанная идея. Обеспечивает баланс между предсказуемыми производственными отклонениями и дискретными макро-рисками, позволяя выстроить устойчивую стратегию опережающей контрактации оборудования (85% до первого бетона).

Место: 9

Почему: Очень прагматичная доработка: сглаживать нужно не абстрактную "толпу", а дефицитные ресурсы. Это позволяет реально оптимизировать мощности ВЗиС и размер жилых городков без угрозы для критического пути.

Место: 11

Место: 13
